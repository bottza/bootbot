A flexible moderation bot for server admins who want free time.

## Installation

```
$ git clone https://github.com/bottza/bootbot.git
$ cd bootbot
$ npm install
```

## Running

One of the following:
```
$ npm start
$ cach node main.js
```

## Development

One of the following:
```
$ npm run watch
$ nodemon main.js
```
